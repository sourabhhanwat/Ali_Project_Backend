# Instruction

## MacOS

1. Install __pyenv__

        brew install pyenv
 
1. Install python

        pyenv install
        
1. Install __poetry__

        brew install poetry
        
1. Install project python dependencies

        poetry install
        
1. Activate python shell, run:

        poetry shell
 
1. To deactivate shell, run:

        exit

1. To start server, run:

        python manage.py runserver
